# Personal AdBlock Filter List

A curated collection of custom adblock filters for personal use.

## Usage

### uBlock Origin
1. Open uBlock Origin dashboard
2. Go to "Filter lists" tab
3. Scroll to "Custom" section
4. Click "Import..." and paste the raw URL of `filters.txt`
5. Apply changes

### Adblock Plus
1. Open Adblock Plus options
2. Go to "Advanced" tab
3. Click "Add filter subscription"
4. Enter the raw URL of `filters.txt`
5. Subscribe to the list

## Local Installation
1. Download `filters.txt`
2. In your adblocker settings, import the file directly

## File Structure
- `filters.txt` - Main filter list in Adblock Plus format
- `README.md` - This documentation

## Filter Categories
- Social media site element hiding
- File hosting site ad removal
- Tracking domain blocking

## Contributing
Add new filters to `filters.txt` following the Adblock Plus syntax.
